## Running calculations

### Tag occurrence 

The `marc-tag-use.py` script simply counts the number (fraction) of records in a given set of input MARCXML records that use a particular MARC tag at least once. On laptop or moderate VM it takes a couple of hours to run:

```
marc-analysis> time python marc-tag-use.py /cul/data/ld4l/bib.xml.full > cornell_marc_usage.dat &

### Record coverage 

The script `marc-record-coverage.py` relies upon a file listing MARC tags in order to use. Here we simply take the output from `marc-tag-use.py` which has tags for the first column, and rename to `marc_tag_order.dat`. The script uses the tag order in this list and runs over all MARCXML data again.

```
> python marc-record-coverage.py /cul/data/ld4l/bib.xml.full/bib.00*.*.xml.gz
